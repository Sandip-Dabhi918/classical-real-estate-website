# Homeverse - Modern Real Estate Platform

![Homeverse Banner](/public/assets/images/logo.png)

Homeverse is a premium, high-performance real estate marketplace built with **Next.js 15**, **Tailwind CSS**, and **Lucide Icons**. It features a stunning, modern design with smooth animations, responsive layouts, and a focus on user experience.

---

## Live Demo

Check out the live site: [Classical Real Estate Website](https://classical-real-estate-website.vercel.app/)

---

## Features

- **High Performance**: Built with Next.js 15 and React 19 for blazing fast page loads.
- **Modern UI/UX**: Clean, professional design with vibrant orange accents and smooth micro-animations.
- **Fully Responsive**: Optimized for all devices—from mobile phones to large desktop screens.
- **Advanced Filtering**: Search for properties by type (Sale/Rent), location, and features.
- **Property Showcases**: Detailed property cards with image galleries, amenities, and agent info.
- **Blog System**: Integrated news and blog section for real estate insights.
- **Location Aware**: Map-ready integration for property locations.
- **Premium Icons**: Uses Lucide React for consistent and crisp vector icons.

---

## Tech Stack

- **Framework**: [Next.js 15](https://nextjs.org/) 
- **Library**: [React 19](https://reactjs.org/)
- **Styling**: [Tailwind CSS 4](https://tailwindcss.com/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Language**: [TypeScript](https://www.typescriptlang.org/)
- **Deployment**: [Vercel](https://vercel.com/)
